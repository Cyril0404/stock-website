## 一、打板客（dabanke.com）详细功能拆解

### 1.2 涨停板页面功能

#### 1.2.1 页面布局

```
┌─────────────────────────────────────────────────────────────┐
│ [涨停板] [跌停板] [炸板股] [强势股]   ← 状态切换标签      │
├─────────────────────────────────────────────────────────────┤
│ 市场：                               时间：              │
│ ○全部 ○沪市 ○深市 ○创业板 ○科创板  ○全部 ○今日  │
├─────────────────────────────────────────────────────────────┤
│ 共找到 67 只股票                                              │
├────┬────────┬──────┬──────┬────────┬────────┬────────────┤
│代码│ 名称   │ 现价 │涨幅%  │ 涨停时间│ 换手率 │ 成交额  │
├────┼────────┼──────┼──────┼────────┼────────┼────────────┤
│... │ ...    │ ...  │ ...  │ 09:32  │ 12.3%  │ 4.2亿    │
└────┴────────┴──────┴──────┴────────┴────────┴────────────┘
```

#### 1.2.2 涨停板表格字段

| 字段名 | 说明 | 数据格式 |
|--------|------|----------|
| 代码 | 股票代码 | 6位数字，如"300992" |
| 名称 | 股票名称 | 中文，如"泰福泵业" |
| 现价 | 当前价格 | 精确到分，如"20.00" |
| 涨幅 | 今日涨幅百分比 | 如"+20.00%" |
| 涨停时间 | 触达涨停的时间 | 精确到分，如"09:32" |
| 换手率 | 当日换手率 | 百分比，如"12.3%" |
| 成交额 | 当日成交金额 | 简化数字，如"4.2亿" |
| 流通市值 | 流通盘大小 | 简化数字，如"32亿" |
| 连板数 | 连续涨停天数 | 数字，如"3"表示3连板 |
| 涨停原因 | 涨停原因标签 | 文字标签，如"商业航天"、"风电" |

#### 1.2.3 涨停分类标签（Tab切换）

| 标签 | 说明 | 筛选条件 |
|------|------|----------|
| 涨停板 | 今日涨停股票 | 涨幅 >= 9.9% |
| 跌停板 | 今日跌停股票 | 跌幅 <= -9.9% |
| 炸板股 | 曾涨停但炸开的股票 | 曾经涨停但当前涨幅 < 9.9% |
| 强势股 | 涨幅超过5%但未涨停 | 5% <= 涨幅 < 9.9% |

#### 1.2.4 市场分类筛选

| 选项 | 说明 |
|------|------|
| 全部 | 沪深京全市场 |
| 沪市 | 上交所主板（60开头） |
| 深市 | 深交所主板（00开头） |
| 创业板 | 创业板（30开头） |
| 科创板 | 科创板（68开头） |
| 北交所 | 北交所（8开头） |

#### 1.2.5 时间筛选

| 选项 | 说明 |
|------|------|
| 全部 | 不限时间 |
| 今日 | 仅显示今日涨停 |

#### 1.2.6 涨停原因标签（多选）

涨停原因来源于股票公告/研报，每个股票可能有多个标签：

| 标签类别 | 示例 |
|----------|------|
| 题材类 | 商业航天、AI智能体、风电、光伏、创新药 |
| 业绩类 | 年报增长、业绩预升、扭亏为盈 |
| 事件类 | 控制权变更、股权转让、中标、资产重组 |
| 行业类 | 机器人概念、新能源汽车、半导体 |
| 地区类 | 地方国资改革 |
